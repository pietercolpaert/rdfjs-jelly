rdfjs-jelly playground

Load a Jelly stream one RDF Message at a time. Plain .jelly and gzip-compressed .jelly.gz URLs are supported. Only the most recent 20 visited messages are retained.

The bundled OSM Denmark 10K stream is selected by default. You can replace it with another URL whose server allows browser CORS access.
